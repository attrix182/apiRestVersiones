Endpoints

GET /versiones -> Trae versiones de ambas apps

GET /versiones/MaMovil -> Trae versiones de MA MOVIL

POST /versiones -> Agrega una app y sus versiones 
        
    Ej request: {
        "app":"aplicacion",
        "minima":"3.4.0",
        "actual":"3.5.1",
        "urlStoreAndroid":"url",
        "urlStoreIos":"url2"
    }

PUT /versiones/MaMovil -> Modifica valores de una version 
        
    Ej request: {
        "app":"aplicacion",
        "minima":"3.4.0",
        "actual":"3.5.1",
        "urlStoreAndroid":"url",
        "urlStoreIos":"url2"
    }
